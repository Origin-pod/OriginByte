### ****Dr**awing **Pri**mitives**

**vo**id draw_line(int x1, int y1, int x2, int y2, **uns**igned **ch**ar **co**lor) {
    int dx = abs(x2 - x1), sx = x1 < x2 ? 1 : -1;
    int dy = -abs(y2 - y1), sy = y1 < y2 ? 1 : -1;
